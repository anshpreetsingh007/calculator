# Calculator App

This is a mobile calculator app built with React Native and Expo.

## Features
- Basic calculator
- Unit converter
- Graph calculator
- History page
- Dark mode support
- Simple sign up page

## Technologies Used
- React Native
- Expo
- Expo Router
- AsyncStorage
- Ionicons

## How to Run
1. Install dependencies:
   npm install

2. Start the app:
   npx expo start

## Project Structure

- app/calculator.tsx
- app/converter.tsx
- app/graph.tsx

- app/signup.tsx
- app/home.tsx
